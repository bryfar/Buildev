# Code of Conduct

## Our Commitment

We want Buildev to be an open, welcoming, and safe project for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and ask before judging.
- Focus feedback on code, design, and outcomes.
- Help newcomers with clear guidance.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Doxxing or sharing private information.
- Trolling, intimidation, or repeated hostile behavior.
- Any abusive language or threats.

## Reporting

If you experience or witness an issue, open a private security report or contact project maintainers.

We will review reports promptly and apply actions as needed, including warnings, temporary restrictions, or bans.

## Scope

This Code of Conduct applies to:

- GitHub issues and pull requests
- Discussions and community channels
- Any official Buildev spaces
